# LAN Device Discovery System
This project implements a decentralised LAN monitoring system that discovers devices using ARP, tracks availability, and sends structured logs to a centralised server. A Streamlit dashboard is used to visualise network activity and device status real time.
## Features
=>ARP-based LAN device discovery
=>Centralized Server for log aggregation
=>Detection of devices entering or leaving the network
=>Display the total number of devices, no, of online and offline devices
=>Real-time monitoring dashboard
=>Lightweight agent archtechture
